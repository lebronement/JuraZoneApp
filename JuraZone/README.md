## Presentation of the app

Cette application vous permet de centraliser tous vos trips et toutes les endroits que vous avez visiter. Vous pourrez donc créer le trips de votre choix. Notre application utilise de la géolocalisation. 

Cette application peut-être utilisée sur toutes type de smartphone, mais également sur tablette ou ordinateur.

Afin de vous retrouver dans notre application, voici la liste des différentes fonctionnalités qui y sont intégrés :

- Création d'un compte
- Login
- Suppression d'un compte
